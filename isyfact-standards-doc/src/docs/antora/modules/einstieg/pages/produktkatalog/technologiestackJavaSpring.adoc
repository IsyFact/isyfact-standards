[[spring-boot-dependencies]]
.Spring Boot Managed Dependencies
****
Das Framework Spring Boot legt selbst zahlreiche Versionen von Drittanbieter-Bibliotheken fest.
Wann immer ein Produkt als Version dieses Icon (icon:leaf[title=Spring Boot Dependency]) enthält, wird die durch Spring Boot festgelegte Version verwendet.

Produkte und ihre Versionen können in der Liste der https://docs.spring.io/spring-boot/appendix/dependency-versions/coordinates.html[Spring Boot Managed Dependencies] nachgeschlagen werden.
****
